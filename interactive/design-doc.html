DESIGN NOTE

高级网页资源库滚动叙事官网

这版网页把本地高级网页资源库整理成一个极简、留白充足、动效丰富的滚动叙事型官网入口，核心目标不是堆链接，而是让每个资源对应一种可以理解和复用的网页能力

资源使用

  • 读取并整合了 resources.json 的 8 个分类，页面底部可按类别筛选查看官网、GitHub 和 npm 信息
  • 把 推荐组合.md 转成 6 个可切换方案，包含适用目标、核心库、注意点和起手命令
  • 把 安装命令.md 转成 6 个可复制命令块，避免用户回到文档里找命令
  • 保留原导航入口，同时把推荐网页转译成页面内可用的动效样本，没有直接嵌入外站

动效系统

  • Lenis 负责平滑滚动，GSAP ScrollTrigger 负责 pinned scroll、parallax、mask reveal 和 image scale
  • SplitType 负责大标题拆字入场，重要文字 hover 时会有小幅位移和下划线反馈
  • 样本区包含滚动手感、滚动时间线、文字拆分、轻 3D 深度、组件灵感、素材展示、图标反馈和视觉气质 8 个样本
  • 磁吸按钮、cursor-follow visual feedback、点击切换详情、资源跳转都服务于浏览和决策，不做无意义装饰

可靠性处理

  • 第三方动效库改为异步加载，页面不会因为 CDN 慢而长时间停在黑屏加载幕布
  • 如果动画库不可用，正文会直接显示，资源筛选、复制、样本切换等基础交互仍可用
  • 移动端导航改为横向可滑，样本卡和资源面板自动回到单列布局
  • 全文已经移除中文句号，改用逗号、分号和自然换行保持阅读节奏

验证结果

已通过 node --check script.js、node --check verification/final-verify.mjs 和 Chrome 真实视口验证 桌面与移动端均生成截图，自动检查了样本卡切换、资源分类跳转、方案切换、资源数量、命令块数量和中文句号清理结果
